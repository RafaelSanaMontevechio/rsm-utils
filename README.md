# Useful functions for working to format Brazilian data

How to use:

```
npm i rsm-utils
```

```
import { function-name } from 'rsm-utils'
```

`function-name` is the name of the function you want to use.

  - maskCEP(value):
    - add a mask to CEP.

  - unMaskCEP(value):
    - remove mask from a CEP.
